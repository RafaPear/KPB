VersionCatalog

data class VersionCatalog(val libs: List<Lib> = listOf(), val versions: List<Version> = listOf(), val plugins: List<Plugin> = listOf())

Represents a Gradle Version Catalog (libs.versions.toml).

Holds collections of libraries, versions, and plugins to be defined in the catalog. Ensures integrity by checking for duplicates.
