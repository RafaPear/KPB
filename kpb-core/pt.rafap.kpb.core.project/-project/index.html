Project

data class Project(val name: String, val group: String?, val versionCatalog: VersionCatalog, val modules: List<Module>, val gradleFiles: List<GradleFile>, val kpbFiles: List<KpbFile>, val templates: List<Template> = emptyList())

Represents a project configuration in the KPB system.

A Project holds all the necessary information to generate a complete project structure on disk, including modules, Gradle build files, version catalogs, and arbitrary files.
