CoreStateSerializer

internal class CoreStateSerializer : Serializer<CoreState, String>

Serializer for the CoreState class, converting it to and from a String representation.

This serializer is used by the storage backend to persist the core state to a file. It handles serialization by converting the state ID to a string, and deserialization by parsing the string back to an integer ID.
